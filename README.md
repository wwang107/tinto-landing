# tinto-landing

Marketing landing page for Tinto, built with React + Vite. Uses the app's
palette (`Color+Claude.swift`) and cropped App Store screenshots from
`tinto-ios/AppStoreScreenshots` (copied into `public/screenshots`).

```bash
npm install
npm run dev       # local dev server
npm run build     # outputs dist/ (relative base — dist/index.html opens directly from disk)
```

Sections: hero · customer features · how check-in works · cafe owner pitch · download CTA · footer (links to `/privacy-policy.html`).
